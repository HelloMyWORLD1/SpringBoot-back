## 개발 환경

### Spring

사이드 프로젝트라 일단은 최신 버전으로 생성하였습니다. (현업에서는 구버전을 쓰기도 합니다.)

샘플코드를 push해 두었습니다.

### DB

DB 접속정보를 입력해야 합니다.

자세한 내용은 `Notion 코드공유`를 참고 부탁드립니다.
